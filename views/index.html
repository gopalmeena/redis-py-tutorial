Redis-py Cheat Sheet

A quick reference for Redis+Python

Instructions before using Commands:

  • First install redis-server in ubuntu by command sudo apt-get install redis-server
  • Now install redis client by command sudo apt-get install redis-cli

    • In python:

  • First install redis by pip install redis.
  • Now use import redis in your (filename).py file or in python shell(which starts with)

    >>>

  • Make a redis connection using r = redis.StrictRedis(host='localhost',port=6379,db=0) . Click here to check all options.
  • You can make a connection Pool too.
